This is the submission for Assignment 1: Creation of a Static Course Website for Data Structure and Programming by Anubhab Chakraborty, Roll No. - 21MCMT34, M.Tech. CS, Semester 2.

The website developed has 1 homepage along with 4 webpages corresponding to 4 modules to be covered in the course. This course has been considered as part of curriculum offered by School of Computer and Information Sciences, University of Hyderabad. So the course related pages describe the modules briefly and mention students to apply to the department office for the complete course.

The website is developed using only HTML and CSS. Some of the images used have been saved locally while for others online link is used to have variety in the project.

To open the homepage open index.html in the browser. The course pages are reachable form the homepage.
